/s_commands(с_команды) - команды на сервере
❗Здесь же названия команд, для изменения их параметров!

❗КОМАНДЫ С АРГУМЕНТАМИ
/изменить_кд <название_команды> <сек>
Пример: /изменить_кд code 0
КД от 0 секунд до 1 000 000 секунд, по умолчанию везде 5 секунд!

/изменить_доступ <название_команды> <0 или 1>
Пример: /изменить_доступ all_updates 0
0 - Отключить
1 - Включить
